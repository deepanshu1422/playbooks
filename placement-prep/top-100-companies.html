üìù Application Strategy

General Tips for All Tiers

1. Resume Optimization

  • Format: ATS-friendly (avoid tables, graphics)
  • Content:
    • Projects with GitHub links (2-3 strong projects)
    • Competitive programming profiles (CodeChef, Codeforces, LeetCode)
    • Internships/relevant experience
    • Technical skills (honest assessment)
    • CGPA (if >7.5, mention prominently)
  • Length: 1 page (strictly)
  • Keywords: Match job description keywords

2. Online Assessments

  • Coding platforms: HackerRank, HackerEarth, Codility, CodinGame
  • Practice: Company-tagged problems on LeetCode/GFG
  • Speed: Time management is critical (solve easy/medium first)
  • Edge cases: Always test with edge cases

3. Referrals

  • LinkedIn: Connect with alumni, employees
  • Cold emails: Polite, personalized approach
  • Timing: Apply with referral early in hiring cycle
  • Follow-up: Gentle reminders after 1-2 weeks

4. Off-Campus vs On-Campus

On-Campus Advantages:

  • Pre-filtered candidates
  • Higher success rate
  • Streamlined process
  • Better packages often

Off-Campus Advantages:

  • Not limited by college tier
  • More companies accessible
  • Can apply year-round
  • Multiple attempts possible

üéì Interview Preparation Tips

Data Structures & Algorithms

Must-Know Topics (Priority Order):

  1. Arrays & Strings (30% of problems)
    • Two pointers, sliding window
    • Prefix sum, Kadane's algorithm
  2. Linked Lists (10%)
    • Reversal, cycle detection, merge
  3. Trees & Graphs (25%)
    • DFS, BFS, Binary trees, BST
    • Shortest path (Dijkstra, BFS)
  4. Dynamic Programming (15%)
    • 1D DP, 2D DP, Knapsack variants
  5. Hash Maps & Sets (10%)
    • Frequency counting, lookups
  6. Stacks & Queues (5%)
    • Monotonic stack, queue implementations
  7. Recursion & Backtracking (5%)
  8. Sorting & Searching (5%)
  9. Heaps/Priority Queues (3%)
  10. Bit Manipulation (2%)

Practice Roadmap:

  • Beginner (0-2 months): 150 Easy + 50 Medium (LeetCode)
  • Intermediate (2-4 months): 100 Medium + 20 Hard
  • Advanced (4-6 months): 50 Medium + 50 Hard + Company-tagged

Resources:

  • Books: Cracking the Coding Interview, Elements of Programming Interviews
  • Platforms: LeetCode, GeeksforGeeks, InterviewBit, HackerRank
  • YouTube: NeetCode, take U forward, Abdul Bari
