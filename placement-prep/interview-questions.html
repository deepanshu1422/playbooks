SQL Questions

  1. Find Second Highest Salary
-- Question: Write query for 2nd highest salary from Employee table
-- Campus: Asked in 60% of interviews

-- Solution 1: Using LIMIT OFFSET
SELECT DISTINCT salary 
FROM Employee 
ORDER BY salary DESC 
LIMIT 1 OFFSET 1;

-- Solution 2: Using subquery
SELECT MAX(salary) 
FROM Employee 
WHERE salary < (SELECT MAX(salary) FROM Employee);
  1. Department-wise Highest Salary
-- Question: Find employees with highest salary in each department
-- Campus: IIT Roorkee 2024

SELECT e.department_id, e.name, e.salary
FROM Employee e
WHERE e.salary = (
    SELECT MAX(salary) 
    FROM Employee 
    WHERE department_id = e.department_id
);
  1. Find Duplicate Emails
-- Question: Find all duplicate emails in Person table
-- Campus: VIT 2025

SELECT email, COUNT(*) as count
FROM Person
GROUP BY email
HAVING COUNT(*) > 1;

üß© Aptitude & Puzzles

Logical Reasoning

1. 100 Doors Problem

Question: "100 doors, 100 people. Person i toggles every ith door. Which doors are open?"

  • Answer: Perfect squares (1, 4, 9, 16, 25, 36, 49, 64, 81, 100)
  • Reason: Perfect squares have odd number of divisors
  • Campus: IIT Bombay 2024

2. Bridge and Torch Problem

Question: "4 people need to cross bridge at night. Max 2 at a time. Times: 1, 2, 5, 10 min. Minimum time?"

  • Answer: 17 minutes
  • Strategy: Fastest person shuttles vs. slowest go together
  • Campus: Asked in OA reasoning section

3. Burning Ropes

Question: "2 ropes, each burns in 60 minutes (non-uniform). Measure 45 minutes?"

  • Answer: Light rope1 both ends + rope2 one end. When rope1 done (30 min), light rope2 other end (15 min more)
  • Campus: BITS Pilani 2025

4. Weighing Balls

Question: "12 balls, 1 defective (heavier/lighter). 3 weighings to find it and determine if heavier/lighter"

  • Classic puzzle: Asked for analytical thinking
  • Campus: NIT Trichy 2024
